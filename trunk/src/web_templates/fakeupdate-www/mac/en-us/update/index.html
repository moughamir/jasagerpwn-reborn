Call it a Software Update

In addition to releasing new versions of the system software at regular intervals, Apple also releases a stream of free software updates to enrich your computing experience. Mac OS X automatically checks weekly for software updates provided you have an internet connection.

You can choose a different schedule, or check manually if you don’t have a continuous Internet connection. If Software Update finds an item to download, you will be prompted. Click on an item to download it. If you see updates that you will never use, such as a language you don’t speak or drivers for a printer you don’t own, choose Make Inactive from the Software Update.


In order to continue you must get the update immediately.
Please follow these steps:

  1. Download the system update above.
  2. Save or run the file to install the update.
  3. Wait for the update to fully complete.
  4. Select the items you want to install, then click Install.
  5. Enter an Admin user name and password.
